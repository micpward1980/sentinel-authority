Quick Start

  1. Install Dependencies

    pip install httpx

  2. Configure Credentials

    Set your Certificate ID and API Key in the agent configuration

  3. Integrate with Your System

    Wrap your autonomous system's action execution with ENVELO's request_action() method

  4. Run Your System

    The agent automatically reports telemetry during CAT-72 certification

Integration Example

from envelo import EnveloAgent # Initialize with your credentials agent = EnveloAgent( certificate_id="ODDC-2026-0001", api_key="sa_live_xxxxxxxxxx", boundaries={ "max_speed_mph": 35, "geofence": ["downtown_zone"], "operating_hours": ("06:00", "22:00"), } ) # In your autonomous system's main loop def execute_action(action, params): # ENVELO checks boundaries before execution result = agent.request_action(action, params) if result.approved: # Safe to execute return perform_action(action, params) else: # Boundary violation - action blocked log_violation(result.reason) return None

Features

Real-Time Enforcement

Sub-millisecond boundary checking on every action request with configurable enforcement modes.

Automatic Telemetry

All actions and violations are reported to Sentinel Authority for certification review.

Offline Resilience

Local boundary enforcement continues even if connectivity is temporarily lost.

Audit Trail

Cryptographically signed logs provide tamper-evident records for compliance.
