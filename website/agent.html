Quick Install

  1. Extract the ZIP and navigate to the folder
  2. Install the package:
    pip install -e .
  3. Get your API key from the Sentinel Authority portal after certification
  4. Configure the agent in your application (see example below)

Quick Start

from envelo import EnveloAgent, EnveloConfig, NumericBoundary # Configure with your ODDC certificate agent = EnveloAgent(EnveloConfig( certificate_id="ODDC-2026-XXXXX", # Your certificate api_key="your-api-key" # From portal )) # Define your ODD boundaries agent.add_boundary(NumericBoundary("speed", max=100, unit="km/h")) agent.add_boundary(NumericBoundary("temperature", min=-20, max=50)) # Wrap autonomous functions - enforcement is automatic @agent.enforce def move_robot(speed, direction): robot.move(speed, direction) move_robot(speed=50, direction="forward") # ✅ Allowed move_robot(speed=150, direction="forward") # ❌ Blocked

Features

Non-Bypassable

Every action must pass through the agent. No exceptions.

Fail-Closed

If the agent fails, all autonomous actions are blocked.

Real-Time Telemetry

All decisions streamed to Sentinel Authority.

Tamper-Evident

Cryptographically signed audit trail.

Multiple Boundaries

Numeric, geographic, rate limits, state checks.

Safe State Callbacks

Trigger emergency procedures on violations.
